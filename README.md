# Homework
一个第三人称射击项目。  
目前就实现了基本的主菜单、基地场景和战斗场景。有一个在基地进行交互进入关卡->完成关卡->对局结束后回到基地的循环。

## 目标
把当前项目做成一个完整的“基地循环 + 关卡战斗 + 结算成长”玩法闭环：  
`主菜单 -> 基地 -> 关卡战斗 -> 结算 -> 返回基地 -> 再次推进`
后面在扩展技能树啊，buff系统啊，新关卡解锁的部分。
再找些美术素材把各个部分美观一下。

## 当前进度

- 有个简单的流程：`MainMenu -> BaseScene_Main -> GameScene -> RunResult -> BaseScene_Main`
- 基地内已具备基础交互入口（战斗入口、强化入口占位）
- 战斗侧已具备基础可玩内容（移动、射击、技能、敌人波次、Boss、结算返回）
- 基本的暂停与返回主菜单界面
